# 📋 Lista de Contatos

> Gerencie seus contatos de forma simples, rápida e organizada.

---

## 🗂️ Sobre

O **Lista de Contatos** é uma aplicação web para cadastrar, visualizar, editar e remover contatos, funcionando como uma agenda digital interativa direto no navegador.

---

## ✨ Funcionalidades

- ➕ Adicionar novos contatos
- 👁️ Listar todos os contatos cadastrados
- ✏️ Editar informações de um contato
- 🗑️ Remover contatos
- 🔍 Busca por nome ou informação
- 💾 Persistência de dados com `localStorage`

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Função |
|---|---|
| 🌐 HTML5 | Estrutura da página |
| 🎨 CSS3 | Estilização e layout |
| ⚡ JavaScript | Lógica e manipulação do DOM |

---

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/Vinizapella/Lista-Contatos.git

# Acesse a pasta
cd Lista-Contatos
```

> Abra o arquivo `index.html` diretamente no navegador — sem necessidade de servidor! 🌐
